# rowing-login

A rowing based login screen with the option to login and signup, signup using google and signup using apple too. Use images of rowing to with 3d effects.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/rowing-login.git
cd rowing-login
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
